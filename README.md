# CareerPilot AI – Agentic Career Counseling Companion

## Overview

CareerPilot AI is an Agentic AI-powered career counseling platform designed to help students and fresh graduates make informed career decisions. The system provides personalized career recommendations, skill-gap analysis, learning roadmaps, certification suggestions, and interview preparation guidance.

Built using **LangFlow, IBM watsonx.ai, and IBM Granite Models**, the solution leverages Retrieval-Augmented Generation (RAG) and a multi-agent architecture to deliver accurate and personalized career guidance.

---

## Problem Statement

Students often struggle to choose the right career path due to limited access to personalized guidance, lack of awareness of industry trends, and uncertainty regarding their skills and interests. Traditional career counseling methods are time-consuming and difficult to scale.

CareerPilot AI addresses these challenges by providing intelligent, data-driven, and personalized career recommendations.

## System Architecture

User → LangFlow → IBM Granite Model → RAG Retriever → Multi-Agent System → Career Guidance Dashboard

### Agents Used

1. Profile Analysis Agent
2. Career Recommendation Agent
3. Skill Gap Analysis Agent
4. Learning Roadmap Agent
5. Interview Preparation Agent

---

## Workflow

1. User provides profile information and career goals.
2. Profile Analysis Agent evaluates skills and interests.
3. RAG Retriever fetches career-related knowledge.
4. IBM Granite Model generates recommendations.
5. Skill Gap Agent identifies missing competencies.
6. Learning Roadmap Agent creates a personalized plan.
7. Interview Preparation Agent provides practice questions.
8. Results are displayed through the dashboard.
